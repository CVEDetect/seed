合作侧审批通知

  1. 请求报文

    1.1 通知方式：POST <Content-Type: application/json; charset=UTF-8>
    1.2 通知频率：每分钟一次，最多五次（15s/15s/30s/3m/10m/20m/30m/30m/30m/60m/3h/3h/3h/6h/6h - 总计 24h4m）
    1.3 通知规则：进件成功或支付失败的单子
    名称 类型 必传 说明
    partnerApplyNo VARCHAR2(32) Y 合作方申请单号
    applyNo VARCHAR2(32) Y 开放平台申请单号
    applyStatus CHAR(1) Y 申请单状态，<取值见附录A：状态码>
    applyRemark VARCHAR2(64) N 申请单审批备注

    [
        {
            "applyNo":"5045133285025547",
            "partnerApplyNo":"6023722639483389",
            "applyStatus":"A"
        },
        {
            "applyRemark":"congratulations",
            "applyNo":"5144245850784466",
            "partnerApplyNo":"9888820839440141",
            "applyStatus":"A"
        }
    ]

  2. 响应报文

    应答字符串“SUCCESS”（大写的）表示收到通知（可能会多次通知，业务侧注意幂等）
